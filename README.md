# Imagine Quest: Enchanted Dugeon!

Oh no, you've become trapped in a dugeon! The only way to escape is to completed the 4 challenges set before you... A subtraction, addition, multiplication room await your arrival. Once complete, push your division skills to the limit and try to beat the Golum! If you can get past all 4 challenges, the dugeon will let you go. Goodluck!

[Enter the Enchanted Dugeon](https://imagine-quest-learning.github.io/IQ_Capstone_Project/)
